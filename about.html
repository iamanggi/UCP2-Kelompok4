PHP

PHP adalah bahasa pemrograman sumber terbuka yang digunakan terutama untuk pengembangan aplikasi web.

HTML

HTML adalah bahasa markup standar untuk membuat dan merancang halaman web.

JavaScript

JavaScript adalah bahasa pemrograman tingkat tinggi dan berbasis web.

Python

Python adalah bahasa pemrograman yang mudah dipelajari dan kuat.

C++

C++ adalah bahasa pemrograman yang digunakan untuk pengembangan perangkat lunak.
